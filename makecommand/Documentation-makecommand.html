Command Name – makecommand

Function – Make and Define a new terminal command

 

Algorithm -

  1. 1.Change directory  to Desktop 

  2. 2.Read name of the command 

  3. 3.Create a bash file with the entered name  

  4. 4.Open this file on a text editor 

  5. 5.Change the file to be executable 

  6. 6.Change the ownership of the file to user 

  7. 7.Run the script to check if the function is performed as expected 

  8. 8.If it does not run as excpected, the text editor opens again 

  9. 9.If it does work as expected, a syntax and a function is read from the user 

  10. 10.These documentation details are then appended to the MyCommands function list. 

  11. 11.The bash file is converted to a extensionless format 

  12. 12.This file is moved to usr/bin
